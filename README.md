matrix-login
=============

a terminal login-manager.

![DEMO](demo.gif)

# Requirements
Python 2 or 3
git

# Instalation
mkdir '~/.config/login'

cd '~/.config/login'

git clone 'https://github.com/eduardoHoefel/matrix-curses'

# Usage
put matrix-login.py to run when you open a terminal

e.g: "echo '~/.config/login/matrix-login/matrix_login.py' > ~/.bashrc"
